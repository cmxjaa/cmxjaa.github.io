# 五. 矛盾
is_numeric遇到%00截断

在URL后添加`?num=1%00`
```
flag{bugku-789-ps-ssdf}
```
### 00截断上传原理
[参考知乎专栏](https://zhuanlan.zhihu.com/p/30005652)

* 截断的核心，就是chr(0)这个字符
* 这个字符不为空(Null)，也不是空字符("")，更不是空格
* 当程序在输出含有chr(0)变量时，chr(0)后面的数据会被停止，换句话说，就是误把它当成结束符，后面的数据直接忽略，这就导致漏洞产生
